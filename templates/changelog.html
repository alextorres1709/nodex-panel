v{{ config.APP_VERSION }}

Version Actual
  • FIX TOASTS ANDROID (DEFINITIVO): Los flash messages seguian apareciendo en la app Android porque el filtro server-side por User-Agent no cubria login.html ni paginas con UA modificada. Ahora MainActivity inyecta CSS+JS en cada pageStarted/pageFinished con un MutationObserver que oculta y elimina cualquier .flash-message aunque se inserte mas tarde.
  • FIX PUSH ANDROID EN RELEASE: Las notificaciones FCM no funcionaban en builds release porque R8 minificaba las clases del SDK de Firebase. Añadidas reglas ProGuard para mantener com.google.firebase.** y los servicios de mensajeria.
  • CANAL FCM PRE-CREADO: Nueva clase NodexApp : Application que crea el canal de notificaciones en el onCreate del proceso, antes que cualquier Activity o Service. Asi las notificaciones background siempre tienen un canal valido aunque MainActivity nunca se haya abierto.
  • BUILD ANDROID: El release ahora se firma automaticamente con el debug keystore via signingConfigs.release. assembleRelease produce un APK instalable directamente. versionCode 1→2, versionName 1.0.0→1.0.1.
Implementado por Alex — 07/04/2026

v4.4.2

  • FIX GOOGLE DRIVE (DEFINITIVO): El bundle del DMG no incluia los 594 ficheros JSON de discovery cache que necesita googleapiclient. Ahora se usa `collect_all` en el spec de PyInstaller para empaquetar todos los datos de googleapiclient, google.auth, google.oauth2, firebase_admin y google.api_core.
  • FIX NOTIFICACIONES REPETIDAS: Las notificaciones nativas de macOS de tareas pendientes saltaban cada 10 segundos — el `last_notified_at` se actualizaba localmente pero el sync pull lo sobreescribia con el valor antiguo del remote. Ahora se hace push inmediato dentro del lock del sync.
Implementado por Alex — 06/04/2026

v4.4.1

  • FIX GOOGLE DRIVE: Primer intento de fix para Drive — añadidos hidden imports al spec de PyInstaller (insuficiente, ver v4.4.2).
  • FIX DELETE DOCUMENTOS: Los documentos eliminados volvian a aparecer tras unos segundos — race condition entre el delete local y el sync pull. Ahora se sostiene el lock del sync durante toda la operacion.
  • FIX DELETE LENTO: Eliminar documentos tardaba muchisimo — la limpieza de Drive se ha movido a un thread en background.
  • FIX UPLOAD MODAL: El modal de subir archivo no se cerraba y permitia subir el mismo archivo varias veces al hacer doble-click — ahora el boton se deshabilita, el modal se cierra y aparece un overlay de carga.
  • FIX PUSH ANDROID: Las notificaciones FCM no llegaban cuando la app Android estaba en background — canal de notificaciones creado en MainActivity.onCreate, icono invalido eliminado.
  • FIX TOASTS ANDROID: Los flash messages del servidor ya no aparecen en la app Android — detectados por User-Agent.
Implementado por Alex — 06/04/2026
